What To Read First

  • Top closed-loop note: 전체 feedback 구조와 lock 의미.
  • Run summary: top CSV에서 뽑은 핵심 숫자.
  • Numeric table: CP/DD는 unsigned, DIFF는 signed로 해석한 표.
  • Event CSV: edge 기준으로 추출한 DLF/hold 이벤트 테이블.
  • SAR integration: CP analog value가 code로 바뀌는 과정.

Current Caution
